This repository contains the source code for **Urdu Assignment Maker**, a web tool for creating Urdu assignments in Nastaliq script. It's open source for transparency and reference related to my website. Primarily for personal use.

Built with basic HTML.

## Features

- Nastaliq font support for Urdu text.
- Custom metadata (institution, subject, student info, date).
- Formatting: bold, italic, underline, highlight, alignment, font size.
- Image upload and embedding.
- PDF export.
- Privacy policy included.
- SEO files: `robots.txt`, `sitemap.xml`.

## Live Site

[pdf-nastaliq.vercel.app](https://pdf-nastaliq.vercel.app/)

## Setup

1. Clone: `git clone https://github.com/creatorsera/urdu-assignment-maker.git`
2. Open `index.html` in a browser.

## How It Works

Enter details, format content, add images, and export to PDF.
