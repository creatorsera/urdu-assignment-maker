This repository hosts the source code for **Urdu Assignment Maker**, a simple web tool for creating Urdu assignments in Nastaliq script. It's open source to demonstrate transparency for my website. This project is primarily for personal use and archival purposes.

Built with basic HTML for a lightweight experience.
